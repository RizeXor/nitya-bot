# nitya-py

A simple discord bot

## Prerequisites

- Python 3.8
- Pip3
- pipenv

## Getting Started

1. `pip3 install pipenv`
2. `git clone https://github.com/RizeXor/nitya-bot.git`
3. `cd nitya-bot`
4. `pipenv install`
5. `pipenv shell`
6. make a file called `.env` in `nitya-py` and paste your token. `TOKEN=YOUR_BOT_TOKEN`
7. `python main.py`
